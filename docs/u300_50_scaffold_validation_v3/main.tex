\section*{Scope}
This report compares \textbf{RNAnneal-ss} against \textbf{RNAstructure}, \textbf{LinearFold-V}, and \textbf{EternaFold} on a \textbf{50-target} subset of the FR3D/BGSU under400 benchmark (truth length $\le 300$), stratified into \textbf{50-nt length buckets}.

\section*{Methods (what differs)}
\begin{itemize}
  \item \textbf{Baselines:} RNAstructure (\texttt{AllSub}), LinearFold-V (\texttt{--zuker}), EternaFold (MEA + \texttt{contrafold sample}).
  \item \textbf{RNAnneal-ss variants:} identical CaCoFold + refinement + MCMC pipeline, but the single-strand scaffold generator (\texttt{Fold} + \texttt{AllSub}-like suboptimals) is swapped:
    \begin{itemize}
      \item \textbf{RNss(RS):} RNAstructure scaffolds.
      \item \textbf{RNss(LF):} LinearFold-V scaffolds.
      \item \textbf{RNss(EF):} EternaFold scaffolds.
    \end{itemize}
  \item \textbf{Duplex sampling:} still uses RNAstructure \texttt{DuplexFold} in all RNAnneal-ss variants (LF/EF do not provide duplex sampling).
  \item \textbf{Seeding:} \texttt{AllSub}-like suboptimal structures from the chosen scaffold backend are injected into the RNAnneal-ss scaffold pool \emph{before} MCMC sampling, and a fixed quota of AllSub-like candidates is retained in the final top-100 to preserve best-of-100 yield.
\end{itemize}
