# Task: Soft Launch (Workstream 2 — RNAnneal Free Tier)

Timeframe: February (target cohort: 10–20 labs)

## Goal

Soft-launch RNAnneal free tier to a small cohort, collect real workloads, and establish an operations + feedback loop.

## Inputs

- MVP build completed (limits/presets/queue/landing page + data policy draft)
- Support/triage workflow (who responds, expected response time, escalation)

## Deliverables

- 10–20 labs onboarded with a repeatable process
- First real job runs collected and categorized (success/failure classes)
- Observability implemented (logs, failures, runtime); basic ops playbook

## Milestones / Steps

- Identify and invite initial labs; schedule short onboarding calls
- Track usage and failure modes; prioritize fixes by impact on reliability
- Set expectations clearly (best-effort, limits, turnaround, contact)
