\section*{Scope}
This report evaluates an \textbf{ensemble} that injects \textbf{LinearFold-V} and \textbf{EternaFold} \textbf{suboptimal} structures into the RNAnneal-ss candidate set, targeting improved \textbf{F1@100} and reduced \textbf{Fail@100}.

\section*{Methods (high level)}
\begin{itemize}
  \item \textbf{RNAnneal-ss (pipeline)}: existing CaCoFold + refinement + MCMC pipeline outputting 100 candidates.
  \item \textbf{Ensemble (RNAnneal-ss + LF + EF)}: merges per-target candidates from RNAnneal-ss, LinearFold-V, and EternaFold, then selects 100 structures with \textbf{length-adaptive per-source quotas} and \textbf{diversity (Jaccard)}.
  \item \textbf{RNAstructure}: \texttt{Fold} MFE structure (padded to 100 for consistent @100 evaluation).
  \item \textbf{LinearFold-V}: \texttt{--zuker} suboptimal enumeration (first 100 used).
  \item \textbf{EternaFold}: MEA prediction plus additional structures from \texttt{contrafold sample} to reach 100 total candidates.
\end{itemize}
